## Git — `<leader>g`

| 키 | 동작 |
|---|---|
| `<leader>gg` | Neogit 열기 (상태/stage/commit/push 통합) |
| `<leader>gc` | 커밋 |
| `<leader>gp` | 푸시 |
| `<leader>gl` | 풀 |
| `<leader>gd` | Diffview 열기 (변경 파일 목록 + diff) |
| `<leader>gh` | 현재 파일 Git 히스토리 |
| `<leader>gq` | Diffview 닫기 |

Neogit 내부 (`<leader>gg`):
| 키 | 동작 |
|---|---|
| `s` | 파일 stage (`S`로 전체 stage) |
| `u` | 파일 unstage |
| `c` → `c` | 커밋 (메시지 작성 후 `:wq`) |
| `p` → `p` | 푸시 |
| `q` | 닫기 |

### Git 워크플로우

```
1. Space g g     Neogit 열기 (상태 확인)
2. s             원하는 파일 stage
3. c → c         커밋 메시지 작성 → :wq 로 저장
4. p → p         푸시
5. q             닫기
```

변경사항 확인:
```
Space g d        변경 파일 목록 + 좌우 diff 비교
Space g h        현재 파일의 Git 히스토리
Space g q        Diffview 닫기
```

## Vim 표준 관례 (커스텀 아님)

### 이동

| 키 | 동작 |
|---|---|
| `gd` | 정의로 이동 (LSP) |
| `gi` | 마지막 편집 위치로 이동 + Insert |
| `K` | 호버 문서 (LSP) |
| `[d` / `]d` | 이전/다음 진단 |
| `gg` / `G` | 파일 처음/끝으로 |
| `w` / `b` | 단어 앞/뒤로 |
| `{` / `}` | 문단 앞/뒤로 |
| `%` | 대응하는 괄호로 |
| `Ctrl+o` / `Ctrl+i` | 이전/다음 위치로 점프 |
| `0` / `$` | 줄 처음/끝 |
| `^` | 줄 첫 글자 |
| `f{char}` / `F{char}` | 현재 줄에서 문자 검색 (앞/뒤) |
| `H` / `M` / `L` | 화면 상단/중간/하단으로 |

### 편집

| 키 | 동작 |
|---|---|
| `i` / `a` | 커서 앞/뒤에서 Insert |
| `I` / `A` | 줄 처음/끝에서 Insert |
| `o` / `O` | 아래/위에 새 줄 Insert |
| `dd` | 줄 삭제 |
| `yy` | 줄 복사 |
| `p` / `P` | 붙여넣기 (아래/위) |
| `u` | 실행 취소 |
| `Ctrl+r` | 다시 실행 |
| `ciw` | 단어 변경 (change inner word) |
| `diw` | 단어 삭제 (delete inner word) |
| `J` | 줄 합치기 (join) |
| `.` | 마지막 명령 반복 |
| `>>` / `<<` | 들여쓰기 / 내어쓰기 |
